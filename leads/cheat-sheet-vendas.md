## Nicho → benefício (1 frase)

| Nicho | Vender assim |
|-------|----------------|
| **Imob** | Fachada + vídeo → portal, IG, TikTok, WhatsApp → **pede visita** |
| **Obra** | Escala + canteiro → redes + investidor |
| **Hotel** | Fachada + entorno → Booking, site, redes → **reserva** |
| **Eventos** | Salão + externa → site, redes, WhatsApp com quem compara |
| **Turismo** | Orla + atrativos → **todas as redes** antes da reserva |

**Pacote sugerido:** imob/hotel/evento → **B** · obra → **B** ou **D** · budget alto → **C**

## Pacotes — faixas sugeridas (zona 1, até 30 km)

> **Sugestões de referência** — não é preço fechado. Proposta com valor certo → Tiago, após endereço.

| Pacote | Inclui | Faixa sugerida |
|--------|--------|----------------|
| **A** | ~15 fotos | R$ 380 – 480 |
| **B** ⭐ | ~15 fotos + vídeo ~45s | **R$ 580 – 720** |
| **C** | ~20 fotos + vídeo 60–90s | R$ 780 – 980 |
| **D** | Obra mensal (mesmo ângulo) | R$ 480 – 620/mês |
| **E** | 3 imóveis, mesmo dia | R$ 1.350 – 1.650 |

Zonas 2–4: ver [guia §4](guia-vendas-comercial.md). **Acima de 150 km:** zona 4 + **~R$ 6/km** (ida) além de 150 → Tiago fecha. Sempre diga *“como sugestão…”* antes da faixa.

## Objeções

| Cliente diz | Responda |
|-------------|----------|
| Quanto custa? | *Como sugestão*, piloto **R$ 580–720** RM; só fotos ~**R$ 380**. Tiago fecha na proposta (2 dias) c/ endereço. |
| É caro | Sugestão: pacote A ou piloto B — costuma caber em 1 anúncio. Valor final → Tiago. |
| Street View | Mapa = rua; drone = lugar **hoje**, exclusivo. |
| Preciso amanhã | Alguns locais exigem autorização (dias). Manda endereço. |
| É legal? | Sim. Tiago valida (ANAC/DECEA/SARPAS). |
| Já tem fotógrafo | Drone complementa: entorno e escala. |
| Manda portfólio | techdrone360.com.br · @techdrone360 |
| Já tenho orçamento | Compare escopo: fotos, vídeo, NF, prazo. Proposta piloto. |
| Mais barato | Ver o que inclui — edição + NF + planejamento de voo. |

**Mais perguntas** (preço, prazo, legalidade, entregas, por nicho): [faq-vendas.md](faq-vendas.md)
