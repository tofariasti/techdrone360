Checklist do dia

Manhã

  • 5–10 contatos novos (Prioridade Alta, Status Pendente)
  • Para cada um: site/IG → gancho → script → enviar → Status Contatado
  • Horário: 9–11h

Tarde

  • Follow-up (3–4 dias sem resposta)
  • Responder interessados (objeções abaixo)
  • Interesse + endereço → template Lead qualificado → Tiago
  • Horário: 14–17h

Multicanal — diga sempre

Material serve em Instagram · Facebook · TikTok · YouTube · WhatsApp · site · Booking · portais

“Um trabalho vira conteúdo para todas as redes — sem contratar de novo para cada canal.”

Antes de enviar (5 min/lead)

  1. Site ou Instagram (coluna Fonte)
  2. 1 gancho real (bairro, obra, salão, roteiro)
  3. WhatsApp ainda no site?
  4. Script do nicho certo

Nicho → benefício (1 frase)

Nicho Vender assim
Imob Fachada + vídeo → portal, IG, TikTok, WhatsApp → pede visita
Obra Escala + canteiro → redes + investidor
Hotel Fachada + entorno → Booking, site, redes → reserva
Eventos Salão + externa → site, redes, WhatsApp com quem compara
Turismo Orla + atrativos → todas as redes antes da reserva

Pacote sugerido: imob/hotel/evento → B · obra → B ou D · budget alto → C

Pacotes — faixas sugeridas (zona 1, até 30 km)

Sugestões de referência — não é preço fechado. Proposta com valor certo → Tiago, após endereço.

Pacote Inclui Faixa sugerida
A ~15 fotos R$ 380 – 480
B ⭐ ~15 fotos + vídeo ~45s R$ 580 – 720
C ~20 fotos + vídeo 60–90s R$ 780 – 980
D Obra mensal (mesmo ângulo) R$ 480 – 620/mês
E 3 imóveis, mesmo dia R$ 1.350 – 1.650

Zonas 2–4: ver guia §4. Acima de 150 km: zona 4 + ~R$ 6/km (ida) além de 150 → Tiago fecha. Sempre diga “como sugestão…” antes da faixa.
